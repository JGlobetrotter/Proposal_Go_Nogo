Proposal Go / No-Go Tool

A web-based decision support tool that helps NGOs and non-profit organizations evaluate whether to pursue a funding proposal. Rather than relying on gut instinct, it provides a structured scoring framework that produces a clear, defensible recommendation: Go, Caution, or No-Go.

Users are NGOs

What It Does
The tool walks evaluators through focused questions across  key dimensions /  categories. As you answer, it calculates a live score and verdict — and lets you export the full assessment as a PDF report.


Assessment Framework: Each of the four dimensions is scored on a 1–5 scale across questions

Dimension Definitions :
◎	Strategic Fit	== Mission alignment, program fit, donor relationships, long-term goals
⬡	  Organizational == Capacity	Staff expertise, preparation time, geographic presence, past performance
◈	Financial Viability ==	Budget adequacy, overhead rates, cash flow, reporting feasibility
△	Risk Assessment ==	Political/security context, scope creep, compliance, reputational exposure


Verdict Thresholds / Scoring :
≥ 70%	== GO /	Proceed with proposal development
50–69% ==	CAUTION	/ Address weak areas before committing
< 50% ==	NO-GO /	Do not pursue this opportunity

Additional Features
    Password-protected access (beta)
    Proposal metadata capture
    Slider-based scoring inputs with real-time calculation
    Section-by-section score breakdown with progress bars
    Color-coded overall verdict (green / yellow / red)
    One-click PDF export with professional formatting

Usage Notes:
    Enter the access password at the login screen
    Fill in the proposal metadata 
    Rate each of the questions using the sliders
    Review your live verdict and section scores on the panel
    Download a PDF report of the full assessment

Tech Stack
    Python 3.11
    Streamlit — web interface
    ReportLab — PDF generation
    Docker / Dev Containers — portable development environment
